dt-net documentation
====================

**dt_net** package are a set of tools to retrieve network type information.

Includes:

.. list-table:: 
   :widths: 25 75
   :header-rows: 1

   * - Package
     - Description
   * - ip_info_helper
     - IP Address information.  Uses ipinfo.io API to retrive IP related information.
   * - net_helper
     - Network utilities helper class.  Information related to retrieving IP, MAC, hostnames, ...
   * - wol
     -  Send WOL packets to target hosts.
